# Security Policy

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please report it responsibly by emailing **support@b20mortgage.com**. 

Please include:
- A detailed description of the vulnerability.
- Steps or a proof-of-concept (PoC) script to reproduce the issue.
- Potential impact of the vulnerability.

We appreciate your help in keeping B20 Mortgage secure. We will acknowledge receipt of your vulnerability report within 48 hours and strive to resolve any verified issues in a timely manner.

*Note: This repository contains educational examples. Do not deploy these contracts to production or use them with real funds.*
